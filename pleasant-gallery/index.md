# 意义不明展览馆

## 群头像展览馆

> 存在于世界树最不起眼根系上的一个节点。由执掌欢愉的大魔法使构建，仅用于存各种乐子。
>
> 其中序号为 0 的展馆就是这里，陈列光锥实验室历史群头像的地方。
